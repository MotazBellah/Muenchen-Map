## Muenchen Map:
This is a single page application featuring a map of munchen, germany. with a list of places I wish to visit, with markers on the map. This app has filter option, so you can search for a places from that list, and by clicking on any place on the list or the marker, wikipedia API and flicker API api provide information about the selected place

## Code style:
This project is written in HTML and JS (MVC).

## Library and Framework:
This app using bootstrap 4 to provides a responsive design for varity of devices. Using KnockoutJS to handle changes the list and filter. Using Jquery and Ajax to get the data from wikipedia and Flicker (3rd party APIs) asynchronously.

## Installation:
Download the project to your local machine.

## Run:
Just open index.html file with one of your best browser and have fun.
